## CSS 写法

```css
/* 隐藏滚动条 */
.hide-scrollbar {
  -ms-overflow-style: none; /* IE and Edge */
  scrollbar-width: none; /* Firefox */
}

.hide-scrollbar::-webkit-scrollbar {
  display: none; /* Chrome, MSEdge, Safari and Opera */
}
```

## LESS 嵌套写法

```less
// 隐藏滚动条
.hide-scrollbar {
  -ms-overflow-style: none; /* IE and Edge */
  scrollbar-width: none; /* Firefox */
  &::-webkit-scrollbar {
    display: none; /* Chrome, MSEdge, Safari and Opera */
  }
}
```
