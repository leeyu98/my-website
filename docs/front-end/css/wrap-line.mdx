## white-space

这个属性是用来设置如何处理元素中<Emphasis>空白字符</Emphasis>的：

- 空白字符是否合并，如何合并。
- 是否换行，如何换行

> 文字换行：指的是文本超出容器的默认自动换行。

:::info 定义

- **空白字符 (spaces)**: 只包括 空格、制表符、换行符<br/>
- **其他空白分隔符 (other space separators)**: 空白字符以外的其他空白字符，如全角空格、零宽空格等

:::

|                | 换行符 | 空格/制表符 | 文字换行 | 行尾空格 | 行尾其他空白分隔符 |
| :------------- | :----- | ----------- | -------- | -------- | ------------------ |
| `normal`       | 合并   | 合并        | 换行     | 删除     | 挂起               |
| `nowrap`       | 合并   | 合并        | 不换行   | 删除     | 挂起               |
| `pre`          | 保留   | 保留        | 不换行   | 保留     | 不换行             |
| `pre-wrap`     | 保留   | 保留        | 换行     | 挂起     | 挂起               |
| `pre-line`     | 保留   | 合并        | 换行     | 删除     | 挂起               |
| `break-spaces` | 保留   | 保留        | 换行     | 换行     | 换行               |

:::warning 注意
`<br />` 元素始终都会导致换行的
:::

## word-break

这个属性指定了在<Emphasis>一个单词的内部</Emphasis>如何换行。

:::info
单词该怎么理解：拉丁字母文字以空格区分单词，中日韩 (`CJK`) 文字不以空格区分单词，可以理解问一个字就是一个单词。
:::

| 属性值         | 作用                                                                                                       |
| :------------- | :--------------------------------------------------------------------------------------------------------- |
| normal         | 完整单词不会被折断，尽可能地独占一行；中日韩文字折行                                                       |
| break-all      | 完整的单词会被折断换行；中日韩文字折行                                                                     |
| keep-all       | 完整的单词不会换行，即使超出容器；中日韩文字也不折行                                                       |
| ~~break-word~~ | <Highlight color="red"> 即将废弃</Highlight> 效果是 `word-break: normal` 和 `overflow-wrap: anywhere` 的合 |

## overflow-wrap

这个属性也是控制单词如何被拆分换行的，实际上是作为 `word-break` 的互补。

:::tip
这个属性原本属于微软扩展的一个非标准、无前缀的属性，叫做 `word-wrap`，后来在大多数浏览器中以相同的名称实现。目前它已被更名为 `overflow-wrap`，`word-wrap` 相当于其别称。
:::
