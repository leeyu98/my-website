基于 `Avatar` 组件封装的头像组件。

![基本用法](./images/avatar-pro.png)

## 特性说明

- 支持多彩模式
- 多词模式支持自动比例缩放 (默认的 `Avatar` 组件存在图像倒置问题)
- 支持 `Avatar` 所有属性

## 基本用法

```jsx {3} showLineNumbers
export default () => {
  return (
    <AvatarPro colorful fontSize={14} size={28}>
      长念
    </AvatarPro>
  );
};
```

## API

> 除此以下附加属性外，支持 `Avatar` 默认所有属性 `AvatarProps`

| 参数     | 类型         | 说明                                                                                      | 默认值                 |
| :------- | :----------- | :---------------------------------------------------------------------------------------- | :--------------------- |
| fontSize | `number`     | <Highlight color="red">必输</Highlight> 字体大小                                          | 12                     |
| children | `string`     | <Highlight>可选</Highlight> 文本，仅支持 `string`                                         | -                      |
| mode     | `AvatarMode` | <Highlight>可选</Highlight> 取字模式，[详见](#avatarmode)                                 | `AvatarMode.FirstChar` |
| colorful | `boolean`    | <Highlight>可选</Highlight> 是否启用多彩显示                                              | `false`                |
| size     | `number`     | <Highlight>可选</Highlight> 头像大小。需要参与字体缩放计算，所以不支持 `AvatarProps.Size` | 28                     |

### AvatarMode

```ts
export enum AvatarMode {
  FirstChar = 'first_char',
  LastChar = 'last_char',
  FullChar = 'full_char',
}
```
