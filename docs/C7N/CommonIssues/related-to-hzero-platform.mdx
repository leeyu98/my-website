## 平台模块初次加载 404

**问题详情**
初次加载模块 `404`，刷新后恢复正常。

**解决方案**

启用初始化加载 `initLoad: true`

> 如果是在其他模块中引用的模块，可以调整正则，在需要时加载

```js title=hzerorc.js
module.exports = {
  packages: [
    { name: 'hzero-front-hiam' },
    { name: 'hzero-front-hmsg', initLoad: true, registerRegex: '\\/hmsg\\/' },
  ],
};
```

## 个性化编码不能通用

`H0` 平台接口请求中包含自定义头信息 `H-Menu-Id`，关联当前菜单 `ID`，存在数据权限控制，所以不同菜单的个性化编码不能通用。
