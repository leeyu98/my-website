法语中一共有 `36` 个音素，其中 `16` 个元音，`17` 个辅音，`3` 个半元音。

:::warning 说明
也有说 `35` 个，是因为 `[a]` 和 `[ɑ]` 读音太接近，不再做区分。
:::

## 元音音素

| 音素 | 发音说明                                              | 发音字母或组合 |
| :--: | :---------------------------------------------------- | :------------- |
| [a]  | 开口较大，舌位靠前，舌微抵下齿                        |                |
| [ε]  | 开口较 [a] 小，舌前部微隆起，舌平抵下齿               |                |
| [e]  | 开口较 [ε] 小，嘴角略向外拉，舌前部微隆起，舌尖抵下齿 |                |
| [i]  | 开口较 [e] 小，嘴角向外拉，舌尖抵下齿，舌前部抬起     |                |
| [u]  | 开口更小，双唇突出呈圆形，舌后缩，舌后部抬起          |                |
| [y]  | 开口极小，双唇突出呈圆形，舌尖抵下齿                  |                |
