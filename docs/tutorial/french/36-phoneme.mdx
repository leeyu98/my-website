法语中一共有 `36` 个音素，其中 `16` 个元音，`17` 个辅音，`3` 个半元音 (也有说 `35` 个，是因为 `[a]` 和 `[ɑ]` 读音太接近，不再做区分)。

:::warning
发音规则是根据大量的语言总结出的规律，所以会有不符合规则的例外情况出现。
:::

## 元音音素

| 音素 | 发音说明                                               | 发音字母或组合                       | 示例       |
| :--: | :----------------------------------------------------- | :----------------------------------- | ---------- |
| \[a] | 开口较大，舌位靠前，舌尖微抵下齿                       | a à â，极少数 emm/enn 前的 e         | [参考](#a) |
| \[ε] | 开口较 \[a] 小，舌前部微隆起，舌尖平抵下齿             | e è ê ë，字母组合 ai aî ei，词末 -et |            |
| \[e] | 开口较 \[ε] 小，嘴角略向外拉，舌前部微隆起，舌尖抵下齿 | é，词末 -er -ez                      |            |
| \[i] | 开口较 \[e] 小，嘴角向外拉，舌尖抵下齿，舌前部抬起     | i î ï 和 y                           |            |
| \[u] | 开口更小，双唇突出呈圆形，舌后缩，舌后部抬起           | ou où oû                             |            |
| \[y] | 开口极小，双唇突出呈圆形，舌尖抵下齿                   | u û                                  |            |

### \[a]

**1. 字母 a 和带有符号的 à â ：**

- `papa` \[papa] _n._ 爸爸
- `sa` \[sa] _adj. poss._ 他/她的
- `tas` \[ta] _m._ 堆，垛，许多
- `las` \[la]
- `à` \[a] _prep._ 到...；向... (与 le, les 连时变 au, aux)
- `âne` \[an] _n._ 驴；笨蛋傻瓜 (词末的 e 不发音)
- `pâle` \[pal] _adj._ 苍白的，微弱的，暗淡的

**2. 极少数 emm/enn 前的 e ：**

- `femme` \[fam] _n. f_ \<阴性名词>女人；妻子

### \[ε]

**1. e 在辅音结尾的闭音节中：**

- `mer` \[mεr] _n._ 海，海水，海洋
- `mec` \[mεk] _n._ 男人；家伙；硬汉
- `sec` \[sεk] _adj._ 干燥的；干旱的；干枯的；生硬的，不柔和的；冷淡的，无情的
- `sel` \[sεl] _n._ 盐

**2. e 在两个相同辅音字母前：**

- `messe` \[mεs]
- `nette` \[nεt] _adj._ 净的，纯的
- `cette` \[sεt] _adj._ 这 (ce 的阴性单数)

**3. 带有符号的 è ê ë ：**

- `sève` \[sεv] _n._ \<植物学>液，汁
- `même` \[mεm]
- `pèle` \[pεl]
- `mère` \[mεr]
- `Noël` \[nɔεl]

**4. ai aî ei 字母组合：**

- `mai` \[mε]
- `aime` \[εm]
- `fraîche` \[frεʃ]
- `naît` \[nε]
- `Seine` \[sεn]
- `neige` \[nεʒ]

**5. 词末的 -et ：**

- `navet` \[navε]
- `filet` \[filε]
- `guichet` \[giʃε]
- `valet` \[valε]

### \[e]

**1. 带有尖音符的 é ：**

- `été` \[ete] _n. m_ \<阳性名词>夏，夏季，夏天
- `né` \[ne]
- `thé` \[te]
- `café` \[kafe] _n._ 咖啡

**2. 词末的 -er -ez ：**

- `visiter` \[vizite]
- `léger` \[leʒe]
- `parler` \[parle]
- `répéter` \[repete]
- `chez` \[ʃe]
- `nez` \[ne]

**3. es 在少数单音节词中 ：**

- `les` \[le]
- `des` \[de]
- `ces` \[se]
- `mes` \[me]

### \[i]

**字母 i 和带有符号的 î ï ：**

- `si` \[si]
- `kaki` \[kaki]
- `qui` \[ki]
- `pile` \[pil]
- `île` \[il]
- `naïf` \[naif]
- `maïs` \[mais] 这里词末的 s 要发音

**字母 y ：**

- `type` \[tip]
- `Fanny` \[fani]

### \[u]

**字母组合 ou où oû ：**

- `loup` \[lu]
- `vous` \[vu]
- `avoue` \[avu]
- `pousse` \[pus]
- `où` \[pus]
- `coût` \[ku]
- `soûle` \[sul]
- `voûte` \[vut]

### \[y]

**字母 `u` 和带有长音符的 `û` ：**

- `tu` \[ty]
- `salut` \[saly]
- `vu` \[vy]
- `sûr` \[syr]
- `mûraie` \[myrε]
- `flûte` \[flyt]

:::warning 容易混淆的发音

- 字母 u 的发音为 \[y]，在单词中一般发\[y]
- 字母 y 的发音为 \[igrεk]，在词中一般发 \[i]
- 音素 \[u] 对应字母组合 ou où oû 的发音
- 音素 \[y] 对应字母 u û 的发音

:::

## 辅音音素

| 音素 | 发音说明                                                        | 发音字母或组合    |
| :--: | :-------------------------------------------------------------- | :---------------- |
| \[f] | 摩擦音，上齿抵下唇，下唇略向内卷；发音时声带不振动              | f ff，字母组合 ph |
| \[l] | 舌尖抵住上齿龈，形成阻塞，气流从舌头两边外出，振动声带          | l ll              |
| \[m] | 双唇合拢，形成阻塞，气流从鼻腔外出，振动声带                    | m                 |
| \[n] | 舌尖抵住上齿龈，形成阻塞，气流从鼻腔外出，振动声带              | n nn              |
| \[v] | 与 \[f] 相对的浊辅音，需要振动声带                              | v                 |
| \[r] | 摩擦音，舌后部抬起，不能卷舌，向软腭与小舌靠近，振动声带 (漱口) | r rr              |

### \[f]

**1. 字母 `f` 和 `ff` ：**

- `flamme` \[flam]
- `difficile` \[difisil]
- `chef` \[ʃεf]
- `fil` \[fil]

**2. 字母组合 `ph` ：**

- `physique` \[fizik]
- `pharmacie` \[farmasi]

### \[l]

**字母 `l` 和 `ll` ：**

- `fil`
- `cil` \[sil]
- `lilas` \[lila]
- `Lille` \[lil]

### \[m]

- `mal` \[mal]
- `mis` \[mi]
- `mille` \[mil]
- `mec` \[mεc]

### \[n]

**字母 `n` 和 `nn` ：**

- `minute` \[minyt]
- `Anne` \[an]
- `Fanny` \[fani]
- `natte` \[nat]
- `nid` \[ni]

### \[v]

- `valise` \[valiz]
- `visite` \[vizit]
- `vitesse` \[vitεs]
- `veine` \[vεn]
- `fève` \[fεv]
- `rivalité` \[rivalite]

### \[r]

字母 r 和 rr ：

- `rare` \[rar]
- `lire` \[lir]
- `race` \[ras]
- `route` \[rus]
- `russe` \[rys]
- `courir` \[kurir]

## 半元音

半元音既有辅音的发音特征，也有元音的发音特征；半元音和与之对应的原因发音方法基本一致，但口型更为紧张；

| 音素 | 发音说明                                               | 发音字母或组合                       | 示例       |
| :--: | :----------------------------------------------------- | :----------------------------------- | ---------- |
| \[j] | 与 \[i] 对应，发音时                                   | i + 元音音素                         | [参考](#a) |
| \[w] | 开口较 \[a] 小，舌前部微隆起，舌尖平抵下齿             | e è ê ë，字母组合 ai aî ei，词末 -et |            |
| \[ɥ] | 开口较 \[ε] 小，嘴角略向外拉，舌前部微隆起，舌尖抵下齿 | é，词末 -er -ez                      |            |

### \[j]

**1. i + 元音音素：**

- `dialogue` \[djalɔg]
- `cahier` \[kaje]
- `clavier` \[klavje]
- `chemisier` \[ʃəmizje]
- `fière` \[fjεr]
- `amitié` \[amitje]

**2. 词首 y + 元音音素 ：**

- `yahourt` \[jaur]
- `yeux` \[jø]
- `yoga` \[yɔga]

**3. 元音音素 + il 在词末；元音音素 + ill ：**

- `travail` \[travaj]
- `bail` \[baj]
- `ail` \[aj]
- `chatouille` \[ʃatuj]
- `bouteille` \[butεj]
- `corbeille` \[kɔrbεj]
- `travailler` \[travaje]
- `brouj` \[brujɔ̃]
- `surveiller` \[syrvεje]

**4. 辅音音素 + ill，读作 \[ij] ：**

- famille \[famij]
- fille \[fij]
- billet \[bijε]
- millet \[mijε]

> 规则的例外情况:

- ville \[vil]
- mille \[mil]
- Lille \[lil]
- village \[vilaʒ]

**5. 辅音群 + i + 元音音素：**

- ouvrier \[uvrije]
- publier \[pyblije]
- crier \[krije]
- oublié \[ublije]
