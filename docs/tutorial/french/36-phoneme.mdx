法语中一共有 `36` 个音素，其中 `16` 个元音 (La voyelle)，`17` 个辅音 (Les consonnes)，`3` 个半元音 (La semi-voyelle)，也有说 `35` 个，是因为 `[a]` 和 `[ɑ]` 读音接近且不易造成歧义，不再做区分)。

> 🍬 发音规则是根据大量的语言总结出的规律，所以会有不符合规则的例外情况出现。

<div className="table__no-header" />

|        |                                                                                                                                                       |
| :----: | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
|  元音  | 口腔元音：`[i]` `[e]` `[ε]` `[a]` `[y]` `[ø]` `[œ]` `[ə]` `[u]` `[o]` `[ɔ]` `(ɑ)` <br/>鼻化元音：`[ɑ̃]` `[ɔ̃]` `[ɛ̃]` `[œ̃]`                              |
|  辅音  | 鼻音：`[m]` `[n]` `[ɲ]` `(ŋ)`<br/>塞音：`[p]` `[b]` `[t]` `[d]` `[k]` `[g]`<br/>擦音：`[f]` `[v]` `[s]` `[z]` `[ʃ]` `[ʒ]` `[r](ʁ)`<br/>近/边音：`[l]` |
| 半元音 | 近/边音：`[j]` `[w]` `[ɥ]`                                                                                                                            |

:::info 法语有没有 `[ŋ]` 这个音？
软腭鼻音 `[ŋ]` 不是法语的固有音位，很多法语母语者也不掌握这个音。
它只出现在外来词的音节尾，例如 parking \[parkiŋ] “停车场”。不掌握该音的人会以硬腭鼻音 `[ɲ]` 替代或发作 `[nɡ]`。
:::

### 鼻化元音

法语中共有 4 个鼻化元音，鼻化元音的发音特点是软腭下降，气流同时从口腔和鼻腔流出，通常对应的是一些以 `-m/-n` 结尾的字母组合，且之后不能有元音字母或者辅音字母 m/n。

| 音素<Placeholder width={40} /> | 发音说明                                         | 发音字母或组合                 | 示例 <Placeholder width={40} /> |
| :----------------------------: | :----------------------------------------------- | :----------------------------- | :-----------------------------: |
|              \[ɑ̃]              | 与 \[a] 对应，发音方式类似，软腭下降             | am an；em en；emm- enn- 在词首 |           [参考](#ε)            |
|              \[ɔ̃]              | 与 \[o] 对应，发音方式类似，软腭下降，舌略向后缩 | om on                          |           [参考](#e)            |
|              \[ɛ̃]              | 与 \[ε] 对应，发音方式类似，软腭下降             | im in ym yn；aim ain eim ein   |           [参考](#a)            |
|              \[œ̃]              | 与 \[œ] 对应，发音方式类似，软腭下降             | 字母组合 um un                 |           [参考](#i)            |

:::info
由于 `[ɛ̃]` 和 `[œ̃]` 的读音较为相似，现代法语中逐渐有被 `[ɛ̃]` 替代的趋势。
:::
