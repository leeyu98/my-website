元音音素是发音的核心与主体部分，所以一般情况下，单词读音中有几个元音音素，就有几个音节。
:::warning
法语单词音节的划分是按照音素来的，而不是字母。
:::

## 音节的划分

**1. 两个元音音素之间的辅音音素属于下一个音节：**

<div className="table__no-header table__no-border table__compact" />

|           |             |         |                                                |
| :-------- | ----------- | :------ | ---------------------------------------------- |
| `valise`  | \[va-liz]   | _n.f._  | 手提箱                                         |
| `magie`   | \[ma-ʒi]    | _n.f._  | 魔法，法术；巫术，妖术；魔术，戏法；魔力，魅力 |
| `lunette` | \[ly-nεt]   | _n.f._  | 望远镜；眼镜                                   |
| `répéter` | \[re-pe-te] | _v. t._ | 重说，重复讲；再一次表达；重做，重新开始       |

**2. 相连的两个辅音音素一般要分开：**

<div className="table__no-header table__no-border table__compact" />

|           |            |     |     |
| :-------- | ---------- | :-- | --- |
| `merci`   | \[mεr-si]  |
| `journal` | \[ʒur-nal] |
| `tourner` | \[tur-ne]  |
| `captif`  | \[kap-tif] |

## 开音节和闭音节

### 开音节

以元音音素结尾的音节；有可能是以 (不发音) 辅音字母结尾。

**1. 以元音字母结尾：**

<div className="table__no-header table__no-border table__compact" />

|        |          |     |     |
| :----- | -------- | :-- | --- |
| `qui`  | \[ki]    |
| `la`   | \[la]    |
| `kaki` | \[ka-ki] |
| `fou`  | \[fu]    |

**2. 以辅音字母结尾：**

<div className="table__no-header table__no-border table__compact" />

|        |       |     |     |
| :----- | ----- | :-- | --- |
| `pas`  | \[pa] |
| `nid`  | \[ni] |
| `coup` | \[ku] |
| `ces`  | \[se] |     |

### 闭音节

以辅音音素结尾的音节；有可能是 (不发音) 元音字母结尾。

**1. 以元音字母结尾：**

<div className="table__no-header table__no-border table__compact" />

|           |            |     |     |
| :-------- | ---------- | :-- | --- |
| `quel`    | \[kεl]     |
| `lac`     | \[lak]     |
| `sic`     | \[sik]     |
| `journal` | \[ʒur-nal] |

**2. 以辅音字母结尾：**

<div className="table__no-header table__no-border table__compact" />

|         |        |     |     |
| :------ | ------ | :-- | --- |
| `sale`  | \[sal] |
| `Seine` | \[Sεn] |
| `pile`  | \[pil] |
| `coupe` | \[kup] |
