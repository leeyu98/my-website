韩语有 21 个元音和 19 的辅音组成 ([在线读音](https://www.2hanyu.com/fayin/biao40.html))。

## 元音

按发音过程中是否改变嘴唇形状和舌头位置，分为 10 个单元音和 11 个双元音。

- 单元音：发音时口型固定无滑动变化；
- 双元音：发音时口型滑动变化。

### 单元音 (10)

|  字母  | 罗马音 | 发音说明                                                      |
| :----: | ------ | ------------------------------------------------------------- |
|   ㅏ   | [a]    | 口型较大，发音类似拼音字母 `a`，发音位置更靠后                |
|   ㅓ   | [eo]   | 开口较小，口型自然。发音类似 `ao`，但发音位置更靠后，口型更小 |
|   ㅗ   | [o]    | 开口更小，口型自然。                                          |
|   ㅜ   | [u]    | 开口度很小，双唇突出呈圆形。发音类似 `wu`                     |
| &#124; | [i]    | 嘴角向两侧拉开                                                |
|   ㅐ   | [ae]   |                                                               |
|   ㅔ   | [e]    | 发音比 `ㅐ` 要小                                              |
|   ㅚ   | [oe]   |                                                               |
|   ㅟ   | [wi]   |                                                               |
|   ㅡ   | [eu]   |                                                               |

:::tip
`ㅐ` `ㅔ` 发音相似，现代韩语已经不做区分，同时 `ㅒ` `ㅖ` 也不做区分。
:::

## 辅音

- 松音：发音口型较自然，松音在词首会发生送气化；
- 紧音：发音口型较紧张，无送气；
- 送气音：发音时有明显送气。

## 收音

韩语文字结构主要有 “上下” “左右” 和 “左右下”，在 “左右下” 结构的文字中位于底部的叫做收音。收音共有 27 个，包括 14 个单收音 (辅音) 和 13 个双收音 (辅音字母组合)。

| 代表音 | 收音                 |  发音  | 发音说明                                                      |
| :----: | -------------------- | :----: | ------------------------------------------------------------- |
|   ㄱ   | ㄱ ㅋ ㄲ ㄳ ㄺ       | `[g]`  | 发音时舌根仅仅顶住软腭，舌尖向下，阻止气流通过 (舌根阻断发音) |
|   ㄴ   | ㄴ ㄵ ㄶ             | `[n]`  | 发音时舌尖一直贴着上齿龈                                      |
|   ㄷ   | ㄷ ㅅ ㅈ ㅊ ㅌ ㅎ ㅆ | `[d]`  | 发音时舌尖紧贴上齿龈，阻止气流通过 (舌尖阻断发音)             |
|   ㄹ   | ㄹ ㄼ ㄽ ㄾ ㅀ       | `[l]`  |                                                               |
|   ㅁ   | ㅁ ㄻ                | `[m]`  | 发音时双唇紧闭，气流从鼻子流出，发出鼻音                      |
|   ㅂ   | ㅂ ㅍ ㅄ ㄿ          | `[b]`  | 发音时双唇阻住气流 (双唇阻断发音)                             |
|   ㅇ   | ㅇ                   | `[ng]` | 发音时舌根顶住软腭，堵住气流，气流从鼻子流出                  |
